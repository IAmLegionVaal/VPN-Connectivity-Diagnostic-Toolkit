# Connectivity Diagnostic Toolkit

A read-only PowerShell toolkit for Windows network connectivity review.

## Features

- Adapter summary
- IP and DNS context
- Route table export
- Remote host TCP tests
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\VPN_Connectivity_Diagnostic_Toolkit.ps1
```

## Safety

Diagnostic-only. It reports network context for support review.
